# Northwind Sales Analytics Dashboard

## Project Overview

This project analyzes the Northwind database using SQL Server and Power BI to uncover insights related to customers, sales, products, suppliers, and employee operations.

## Project Files

### Documentation
- Northwind_MECE_Framework.docx
- NORTHWIND_EDA.xlsx
- Northwind_Sales_Analysis_ppt.pptx

### Dashboard
- NORTHWIND_ANALYSIS_DASHBOARD.pbix

## Analysis Areas

### Customer & Order Analysis
- Customer Segmentation
- Order Frequency
- Geographic Analysis
- Revenue Contribution

### Product & Supplier Analysis
- Product Performance
- Category Analysis
- Supplier Distribution
- Pricing Analysis

### Employee & Operations Analysis
- Employee Distribution
- Employee Tenure
- Shipping Performance
- Order Management
