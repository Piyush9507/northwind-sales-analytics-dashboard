# Northwind Sales Analytics Dashboard

## Project Overview
This project analyzes the Northwind database using SQL Server, Excel, and Power BI to generate business insights related to customers, orders, products, suppliers, and employee operations.

## Files Included
- Northwind_MECE_Framework.docx
- NORTHWIND_EDA.xlsx
- NORTHWIND_ANALYSIS_DASHBOARD.pbix
- Northwind_Sales_Analysis_ppt.pptx

## Dashboard Pages
1. Executive Summary
2. Customer & Order Analysis
3. Product & Supplier Analysis
4. Employee & Operations Analysis
